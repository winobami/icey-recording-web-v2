# Icey Recording · Soulflow Movement

> A culture for the outsiders.

**Icey Recording** · Est. 2009 · Los Angeles

---

## About

Official website for Icey Recording LLC and the Soulflow Movement. Built with a minimalist, dark, cinematic aesthetic — black, white, and faint purple haze.

## Tech Stack

- **Next.js 14** (App Router, Static Export)
- **TypeScript**
- **Tailwind CSS**

## Sections

- **Hero** — Centered logo, headline, radial purple haze background with vertical hairlines
- **Artists** — Win Obami bio, portrait, social links
- **Manifesto** — Soulflow Movement manifesto
- **Video Spotlight** — Featured YouTube embed
- **Footer** — Label info, Instagram link

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Build

```bash
npm run build
```

Static export outputs to `/out`.
